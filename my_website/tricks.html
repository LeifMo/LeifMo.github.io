Tip: CSS can be used to style your links. For example, in the case of the links to the other pages on this site above, I used the following code:

a.class1:link, a.class1:visited {

background-color: rgb(224,224,224);

color: black;

padding: 1px 1px;

text-align: center;

text-decoration: none;

display: inline-block;

}

a.class1:hover, a.class1:active {

background-color: lightblue

}

This allowed me to give those links specific backgrounds that change color when scrolled on. You may have also noticed how I wrote class. This brings me to my next tip:

Tip: You can categorize elements of the CSS into classes.

Essentially, by specifying a class, you make it so those links only follow the rules specified in that specific CSS class set.
